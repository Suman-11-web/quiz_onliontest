# Quiz Questions Website

A modern and interactive Quiz Questions Website built using HTML, CSS, JavaScript, and Python backend technologies. This project allows users to answer quiz questions in a clean and user-friendly interface while improving knowledge and learning experience.

# Features

- Multiple choice quiz questions
- Interactive question interface
- Score calculation system
- Modern responsive UI design
- Fast and lightweight website
- Mobile-friendly layout
- Dynamic question display
- User-friendly navigation
- Simple and clean design
- Beginner-friendly project structure

# Technologies Used

- HTML5
- CSS3
- JavaScript
- Python
- GitHub
- Vercel Hosting

# Project Purpose

The main purpose of this project is to practice frontend and backend web development concepts while creating a useful educational platform. This project also helped improve JavaScript logic, website design skills, deployment experience, and backend integration knowledge.
